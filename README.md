# Smart Chair System

智能座椅系统 - 集成云端服务、模拟器和移动APP的完整解决方案

\# 智能座椅系统



这是一个智能座椅的控制系统，包含：

1\. 手机APP（控制端）

2\. 座椅模拟器（模拟真实座椅）

3\. 云端服务器（数据存储和AI分析）



\## 快速开始

1\. 启动服务器

2\. 启动模拟器

3\. 用手机APP连接



\## 项目结构

\- backend/ - 服务器代码

\- simulator/ - 座椅模拟器

\- shared/ - 共享代码

## 项目结构

* backend/: 云端服务端
* simulator/: 座椅模拟器
* shared/: 共享代码和协议
* deployment/: 部署配置
* docs/: 项目文档
* scripts/: 开发脚本
